# BloodCore

BloodCore is a fast-paced arena shooter prototype made with Unity.
Inspired by classic shooters like Doom, the player fights against demon cube enemies in a dark arena while managing movement, ammo and positioning.

Created by Fabian Guz.

## Features

* Arena-style combat
* Enemy spawning system
* Demon cube enemies
* Shooting and recoil mechanics
* Ammo and reload system
* Blood effects and decals
* Health and ammo pickups
* Score system
* Main menu and pause menu
* Atmospheric lighting and fog

---

## Controls

* W A S D – Move
* Mouse – Look / Aim
* Left Click – Shoot
* R – Reload
* ESC – Pause Menu
