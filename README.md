# BloodCore

BloodCore is a fast-paced arena shooter prototype made with Unity.

Inspired by classic shooters like **Doom**, the player fights against demon cube enemies inside a dark arena while managing movement, ammo and positioning.

Created by **Fabian Guz** and **Lolo**.

# Features

### Core Gameplay
- Arena-style combat
- Enemy spawning system
- Demon cube enemies
- Score system
- Blood effects and decals
- Atmospheric lighting and fog

### Weapons
- Multiple weapons:
  - Pistol
  - Uzi (SMG)
  - Benelli M4 (Shotgun)
  - AK74
- Weapon selection menu before starting the game
- Weapon-specific recoil balancing
- Shotgun pellet spread system
- Weapon specific muzzle flash and muzzle light effects

### Combat Systems
- Shooting and recoil mechanics
- Ammo and reload system
- Randomized pickups:
  - Ammo pickups give **25–50 ammo**
  - Health pickups give **1–3 health**

### Movement
- First-person movement
- Dash ability

### UI & Systems
- Main menu
- Pause menu
- Settings menu
- Volume slider
- Mouse sensitivity slider
- FPS limiter
- Fullscreen toggle
- Settings saved between sessions

# Controls

W A S D — Move  
Left Shift — Dash  
Mouse — Look / Aim  
Left Click — Shoot  
R — Reload  
ESC — Pause Menu  
F11 — Toggle Fullscreen
